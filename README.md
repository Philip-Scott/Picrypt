##Picrypt: Encrypt and Decrypt into images
####By: Felipe Escoto



### Install: 
*	$ cd [to-this-directory]
*	$ cc     Picrypt.c   -o picrypt -std=c99
*	$ sudo cp picrypt /usr/bin/
	
### Basic usage:
 * Encrypt: $ picrypt [FILE].xyz
 * Decrypt: $ picrypt [IMAGE].bmp
 
### Advanced usage:
 * $ picrypt [IMAGE].bmp [FILE].xyz
